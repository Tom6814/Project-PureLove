## 📖 序言：纯爱的灯塔

在这个充斥着快餐式感官刺激与机械化情欲的时代，**纯爱（Vanilla/Wholesome Romance）** 宛如被泥沙掩埋的稀世珍宝。人们迷失在无尽的猎奇标签与空洞的肉体关系中，渐渐忘却了指尖相触时的悸动、眼神交汇时的温情，以及灵魂共鸣时那种最原始、最纯粹的爱意。

**「纯爱计划」(Project PureLove)** 应运而生。

这不是一个普通的漫画聚合站。这是一个神圣的避难所，一座专为纯爱战士们建立的圣殿。我们致力于从浩如烟海的 JM 漫画库中，由人工与信仰共同筛选出那些真正拥有**高画质、剧情唯美、情感细腻**的诚意之作。

在这里，没有背叛，没有绝望，只有**相互救赎、青梅竹马、新婚燕尔**的甜蜜羁绊。每一部被收录的作品，都经过了纯爱审判庭的严格洗礼，确保它能为您带来最真挚的心灵慰藉。

## ✨ 核心信条与功能

### 1. 圣洁的准入机制 (Sacred Submission)
任何信徒都可以通过提供 **JM ID** 向上帝（管理员）提交您心中的纯爱神作。
* **智能解析**：提交瞬间，系统将通过我们重铸的 API 接口，瞬间剥离防盗链与阻碍，将封面图化为永恒的 Base64 圣印存入殿堂。
* **隔离感官**：官方原本粗糙的简介与信徒们真挚的「阅读感想/推荐语」被严格分离，确保后来者能先感受到推荐者的炽热情感。

### 2. 纯爱审判庭 (The Tribunal of Love)
每一部提交的作品都将进入“炼狱”（Pending 队列）。只有经过具有极高纯爱信仰的 **管理员 (Admin)** 或 **审判官 (Reviewer)** 严格审核，确认其内容无毒、无雷、无牛头人（NTR）后，方可被批准进入**「本周纯爱推荐」**的万神殿。

### 3. 信仰共鸣系统 (Community of Believers)
* **不可亵渎的评价**：为了保持圣殿的纯洁，我们谢绝了一切匿名访客的喧哗。唯有登录注册的信徒，方有资格在作品下留下星级与感悟。
* **唯一真言**：每位信徒对同一神作只能留下一次真诚的评价，修改与更新将精确影响作品的综合星级（★），确保评分绝对公正、不容操纵。
* **身份徽记**：信徒可于个人设置中雕刻自己的显示名与 JM 专属代号，以最虔诚的姿态与同好交流。

### 4. 直达彼岸 (The Bridge to Elysium)
不再需要繁琐的复制粘贴，每一部获批的神作详情页都刻有**「前往观看」**的传送法阵，点击即可直达禁漫的彼岸，开启灵魂的洗礼。

## 🛠️ 圣殿基石 (Tech Stack)

这座圣殿建立在现代前端魔法与无服务器云端基石之上，坚不可摧：

* **核心框架**: `React 19` + `Vite`
* **圣衣编织**: `Tailwind CSS 4` + `Framer Motion` (极致流畅的交互与转场动画)
* **信仰之门 (Auth)**: `Firebase Authentication`
* **记忆石板 (Database)**: `Firebase Firestore` (实时同步评分、评论与审核状态)
* **数据引流 (Backend)**: `Express` + `Node.js` (搭载强力的 AES-256-ECB 解密核心与并发 CDN 探针)
* **云端王座**: 完美适配 `Zeabur` 的 Docker 全栈容器化部署
