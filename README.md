## 📖 序言：纯爱的灯塔

在这个充斥着快餐式感官刺激与机械化情欲的时代，**纯爱（Vanilla/Wholesome Romance）** 宛如被泥沙掩埋的稀世珍宝。人们迷失在无尽的猎奇标签与空洞的肉体关系中，渐渐忘却了指尖相触时的悸动、眼神交汇时的温情，以及灵魂共鸣时那种最原始、最纯粹的爱意。

**「纯爱计划」(Project PureLove)** 应运而生。

这不是一个普通的 JM 漫画分享站。这是一个神圣的避难所，一座专为纯爱战士们建立的圣殿。我们致力于从浩如烟海的 JM 漫画库中，由人工与信仰共同筛选出那些真正拥有**高画质、剧情唯美、情感细腻**的诚意之作。

在这里，没有背叛，没有绝望，只有**相互救赎、青梅竹马、新婚燕尔**的甜蜜羁绊。每一部被收录的作品，都经过了社区的严格把关，确保它能为您带来最真挚的心灵慰藉。

## ✨ 核心理念与功能

### 1. 严格的准入机制 (Strict Submission)
任何热爱纯爱的读者都可以通过提供 **JM ID** 向社区推荐您心中的佳作。
* **智能解析**：提交瞬间，系统将通过我们的后端 API 接口，直接剥离防盗链，将高质量封面图转换为 Base64 数据永久存入社区。
* **分离感想**：官方的客观简介与读者们真挚的「阅读感想/推荐语」被严格分离，确保后来者能更直观地感受到推荐者的真情实感。

### 2. 纯爱守护庭 (The Guardian Tribunal)
每一部提交的作品都将进入待审核（Pending）队列。只有经过热爱纯爱的 **管理员 (Admin)** 或 **审核员 (Reviewer)** 严格把关，确认其内容无毒、无雷、无牛头人（NTR）后，方可被批准进入**「本周纯爱推荐」**的聚合页。

### 3. 纯洁的共鸣社区 (Community Resonance)
* **负责任的评价**：为了保持社区的纯粹，我们谢绝了匿名访客的随意灌水。唯有注册登录的读者，方有资格在作品下留下星级与感悟。
* **客观公正**：每位读者对同一部作品只能留下一次真诚的评价，修改与更新将精确影响作品的综合星级（★），确保评分绝对公正、不容操纵。
* **个性身份**：读者可于个人设置中编辑自己的显示名与 JM 专属代号，以最舒适的姿态与同好交流。

### 4. 直达彼岸 (Direct Access)
不再需要繁琐的复制粘贴搜索，每一部获批的佳作详情页都设有了**「前往观看」**的直达按钮，点击即可直通禁漫的详情页。

## 🛠️ 圣殿基石 (Tech Stack)

这座圣殿建立在现代前端魔法与无服务器云端基石之上，坚不可摧：

* **核心框架**: `React 19` + `Vite`
* **圣衣编织**: `Tailwind CSS 4` + `Framer Motion` (极致流畅的交互与转场动画)
* **信仰之门 (Auth)**: `Firebase Authentication`
* **记忆石板 (Database)**: `Firebase Firestore` (实时同步评分、评论与审核状态)
* **数据引流 (Backend)**: `Express` + `Node.js` (搭载强力的 AES-256-ECB 解密核心与并发 CDN 探针)
* **云端王座**: 完美适配 `Zeabur` 的 Docker 全栈容器化部署

## 📜 社区公约 (The Covenant)

1. **绝对纯爱**：绝不提交任何包含 NTR、血腥、猎奇、精神控制等不符合纯爱标签的作品。
2. **真诚交流**：在评论区保持友善与尊重，用分享的心态与他人交流。
3. **捍卫净土**：若发现有伪装成纯爱的不适作品混入，请立刻联系管理员将其下架。
